# Proyectos destacados

## Evaluación e implementación de WAF para pymes

Investigación experimental enfocada en la selección e implementación de un Web Application Firewall (WAF) de código abierto sobre hardware no dedicado para entornos pyme con recursos limitados.

### Resultados principales

* 99.29% de tasa de detección
* 58.37% menos falsos positivos
* 4.90% de consumo promedio de CPU
* 0.03 GiB de RAM promedio
* Dataset de 134.092 peticiones HTTP reales

### Tecnologías

`Coraza WAF` • `OWASP CRS` • `Docker` • `Python` • `Scikit-Learn` • `Prometheus` • `Grafana`

🔗 **[Ver organización del proyecto](https://github.com/Web-Application-Firewall)**

## Arquitectura Cloud-Native Multi-Región en Azure

Diseño e implementación de una arquitectura cloud-native desplegada sobre múltiples regiones de Azure, orientada a alta disponibilidad, resiliencia y automatización.

### Aspectos destacados

* Infraestructura como Código con Terraform
* GitOps con ArgoCD
* AKS multi-región
* Failover automático
* Service Mesh con Linkerd y mTLS
* Observabilidad integral
* Seguridad automatizada dentro del pipeline CI/CD

### Stack tecnológico

`Azure` • `AKS` • `Terraform` • `Kubernetes` • `Helm` • `GitHub Actions` • `ArgoCD` • `Linkerd` • `Redis` • `Prometheus` • `Grafana` • `Loki` • `Tempo`
